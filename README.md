# A股实时监控台

实时行情监控面板，支持群兴玩具(002575) vs 奥瑞德(600666) 对比分析。

## 功能

- 实时行情（腾讯财经免费接口）
- 五档行情（买一至买五 / 卖一至卖五）
- 分时行情图（Recharts）
- 板块共振分析
- 持仓盈亏计算
- 操作信号（强烈买入 / 买入 / 持有 / 减仓 / 卖出）
- 支撑位 / 压力位 / 动量 / 委比
- 自动刷新（每30秒）+ 手动刷新

## 快速开始

```bash
npm install
npm run dev
```

访问 http://localhost:3003

## 技术栈

Next.js 14 · TypeScript · Tailwind CSS · Recharts
